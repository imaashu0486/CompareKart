🚀 QUICK START (60 SECONDS)
═══════════════════════════════════════════════════════════════════════════

  Windows:
  ─────────
  cd backend && setup.bat && venv\Scripts\activate.bat && python main.py
  
  In another terminal:
  cd frontend\comparekart_app && setup.bat && flutter run

  Mac/Linux:
  ──────────
  cd backend && ./setup.sh && source venv/bin/activate && python main.py
  
  In another terminal:
  cd frontend/comparekart_app && ./setup.sh && flutter run

  Then:
  ──────
  ✓ Backend runs on: http://localhost:8000
  ✓ API docs at: http://localhost:8000/docs
  ✓ App opens in: Flutter emulator/device
